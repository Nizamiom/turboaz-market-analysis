# Turbo.az Market Analysis
Exploratory Data Analysis of 650K+ automobile listings using Python and Power BI.

## Project Overview

This project analyzes over 650,000 automobile listings from Turbo.az to identify the factors that influence listing popularity and user engagement.

## Objectives

- Clean large datasets
- Perform Exploratory Data Analysis
- Identify market trends
- Visualize business insights
- Build an interactive Power BI dashboard

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- SQL

## Key Findings

- Mercedes and Toyota consistently received more views than Hyundai and Kia despite similar VIP promotion rates.
- Brand reputation had a stronger influence on engagement than paid promotion.
- Longer descriptions and richer equipment lists correlated with higher listing views.
